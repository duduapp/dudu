# 嘟嘟

基于Flutter开发的mastodon长毛象客户端，支持安卓和iOS

## 功能

多账号  
暗色模式  
翻译嘟文  
创建投票  
展示网站自定义emoji  
浏览其它站点的公共时间轴  
...


## 安装

这个项目基于Flutter **1.22.6**, 你可以使用 [fvm](https://github.com/fluttertools/fvm) 来管理Flutter版本.
如果你已经装后了fvm,执行下面代码。

```shell
git clone https://github.com/duduapp/dudu
cd dudu
fvm install 1.22.6
fvm flutter pub get
fvm flutter run
```
如果已经启动了安卓或iOS模拟器，App应该就会运行。
